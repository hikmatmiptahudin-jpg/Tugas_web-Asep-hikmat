# Tugas Lab Pemrograman Web: Portofolio Digital

## Deskripsi Proyek
Proyek ini adalah aplikasi web sederhana berbasis **HTML, CSS, dan JavaScript** yang berfungsi sebagai kartu identitas digital interaktif. Fokus utama tugas ini adalah pada implementasi manipulasi DOM dan penanganan event.

### Fitur Utama
- **Dark Mode**: Fitur interaktif untuk mengubah tema tampilan kartu secara dinamis.
- **Event Listener**: Implementasi alert saat foto profil diklik.
- **Responsive Design**: Tampilan yang menyesuaikan untuk perangkat mobile (HP).

### Teknologi yang Digunakan
- **HTML5**: Sebagai struktur dasar halaman web.
- **CSS3**: Digunakan untuk tata letak (*Flexbox*), desain kartu, dan mode gelap.
- **JavaScript**: Untuk logika interaksi dan manipulasi elemen DOM.

### Cara Menjalankan
1. Pastikan semua file (`index.html`, `style.css`, `script.js`, dan `foto.jpg`) berada dalam satu folder.
2. Buka file `index.html` menggunakan browser pilihan Anda.
3. Gunakan tombol `Ubah Mode Tampilan` untuk mencoba fitur JavaScript.
